# MERN Auth (Simple)

## Run Backend
```bash
cd backend
npm install
npm run dev
```

## Run Frontend
```bash
cd frontend
npm install
npm start
```

- Backend: http://localhost:5000
- Frontend: http://localhost:3000 (proxy to backend)
- Default Mongo URI: mongodb://127.0.0.1:27017/authDemo
